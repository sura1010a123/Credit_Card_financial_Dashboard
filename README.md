📊 Financial Dashboard using Power BI
🔹 Project Overview

This project presents a Financial Dashboard built using Power BI to analyze and visualize financial data from a CSV file. The dashboard provides key insights into revenue, expenses, profit trends, and overall business performance.

The goal of this project is to demonstrate data analysis, visualization, and business intelligence skills using tools like Power BI.

🔹 Dataset
Source: CSV file (financial data)
Contains:
Date
Revenue
Expenses
Profit
Category / Segment
🔹 Tools & Technologies Used
Power BI
CSV (Data Source)
DAX (Data Analysis Expressions)
Data Cleaning & Transformation
🔹 Key Features
📈 Revenue, Expense, and Profit Analysis
📊 Interactive Dashboard with Filters & Slicers
📅 Time-based Analysis (Monthly / Yearly trends)
💡 KPI Indicators for business performance
🔍 Category-wise financial breakdown
🔹 Key Insights
Identified high and low revenue periods
Analyzed profit trends over time
Compared expenses across categories
Highlighted key business performance metrics
🔹 DAX Measures Used

Some of the important DAX calculations used:

Total Revenue = SUM(Revenue)
Total Expenses = SUM(Expenses)
Profit = Revenue - Expenses
Profit Margin %
